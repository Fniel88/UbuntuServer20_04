Tipos de Enrutamiento

  • Estático
  • Dinámico
  • Predeterminado

Enrutamiento Estático

Es configurado manualmente por el administrador de red. Es útil en redes pequeñas.

  • ✔ Más seguro
  • ❌ Poco flexible
  • Ejemplo: `ip route add 192.168.1.0/24 via 10.0.0.1`

Enrutamiento Dinámico

El router aprende las rutas automáticamente usando protocolos como RIP, OSPF o BGP.

  • ✔ Adaptativo
  • ❌ Más complejo
  • Protocolos comunes: RIP, OSPF, BGP

Enrutamiento Predeterminado

Se usa cuando no se conoce una ruta específica. Es el "camino por defecto".

  • ✔ Simplicidad
  • ❌ Puede saturarse
  • Ejemplo: `ip route add default via 192.168.0.1`
